ViPRuby

### A Ruby library for EMCs ViPR Controller (ViPR-C) REST API

How to use:

Install and usage:

gem install vipruby  
require 'vipruby'

Create a ViPR object:

base_url = 'vipr.mydomain.com'
user_name = 'root'
password = 'mypw'
verify_cert = false

vipr = Vipr.new(base_url,user_name,password,verify_cert)
